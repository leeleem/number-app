# 数当てゲーム

0～99の数字を当てるシンプルなゲームアプリです。

## 📱 特徴

- **シンプルで楽しい** - 難しい操作は一切なし
- **最高記録を保存** - あなたの最速記録が残る
- **スマホ対応** - スマートフォンでホーム画面に追加できる
- **オフライン対応** - インターネット接続不要（保存機能のため初回アクセス時のみ必要）

## 🎮 ゲームのルール

1. 0～99の数字を1つ入力
2. 「判定」ボタンを押す
3. 「大きすぎます」「小さすぎます」のヒントが表示される
4. 正解するまで繰り返す
5. 試行回数が少ないほど成績が良い

## 🚀 使い方

### Web版（ブラウザ）
1. `index.html`をブラウザで開く
2. ゲームを楽しむ

### スマホアプリ版
#### iOS（Safari）
1. SafariでこのアプリのURLを開く
2. 下のメニュー → 「ホーム画面に追加」
3. ホーム画面にアプリが追加される

#### Android（Chrome）
1. Chromeでこのアプリを開く
2. メニュー（⋮） → 「ホーム画面に追加」
3. ホーム画面にアプリが追加される

## 🎯 主な機能

### 1. ゲーム開始
- 「新しいゲーム」ボタンで何度でもプレイ可能

### 2. 判定機能
- 入力した数字と答えを比較
- ヒントを表示
- Enterキーでも判定可能

### 3. 試行回数カウント
- 何回で当てたかを記録

### 4. 最高記録保存
- localStorageを使用して保存
- ブラウザのデータ削除まで残る

## 🎨 カスタマイズ

### 難易度を変更したい場合

`script.js`の以下の行を変更：

```javascript
// 現在：0～99
answer = Math.floor(Math.random() * 100);

// 0～999にする場合
answer = Math.floor(Math.random() * 1000);

// 0～9にする場合
answer = Math.floor(Math.random() * 10);

```

### 色を変更したい場合
`style.css`の以下の部分を変更：

```css
/* グラデーション背景の色 */
background: linear-gradient(135deg, #667eea 0%, #764ba2 100%);

/* ボタンの色 */
background: #667eea;
```
